# 🔒 Password Generator

A simple, responsive web-based **Password Generator** built using vanilla HTML, CSS, and JavaScript. This tool allows users to generate strong, randomized, and secure passwords based on customizable criteria.

## 🚀 Features
* **Customizable Length:** Set password lengths from 4 to 30 characters.
* **Character Filters:** Choose to include or exclude uppercase letters, numbers, and special symbols.
* **Real-time Validation:** Alerts you if no character sets are selected.
* **Clean Dark Mode UI:** Modern, responsive design that looks great on both desktop and mobile screens.

## 🛠️ Tech Stack
* **HTML5** - Structure and layout
* **CSS3** - Styling and custom dark-theme UI
* **JavaScript (ES6)** - Password generation logic and DOM manipulation
