# Command Line Blackjack

A minimalist, object-oriented Blackjack game built natively in Python. This project demonstrates clean coding practices, Object-Oriented Programming (OOP) principles, and basic state-machine game design in Python.

## 🚀 Features
- Fully object-oriented layout (Cards, Decks, Hands, and Game logic are isolated).
- Automatic soft/hard Ace calculation (evaluates to 11 or 1 dynamically depending on context).
- Input validation to prevent crashes.
- Clean and interactive ASCII command-line interface.
